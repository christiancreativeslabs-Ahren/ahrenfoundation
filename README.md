# Ahren Foundation Website

> **Tech Skills. Kingdom Purpose. Global Impact.**

A modern, animated website for Ahren Foundation — a platform that empowers youths for societal impact and purposeful living through tech programs, mentorship, and strategic collaborations.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home — Hero, stats, programs overview, vision, CTA |
| `/about` | About — Vision, mission, anchor, beliefs |
| `/what-we-do` | Programs — All 6 initiatives in detail |
| `/team` | Team — Executive team with photos |
| `/blog` | Blog — Articles with tag filtering & modal reader |
| `/join` | Join — Dual signup forms (Youth + Mentor) |
| `/contact` | Contact — Contact info + contact form |

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deployment

Ready for deployment on **Vercel** (recommended) or any Node.js hosting platform.
